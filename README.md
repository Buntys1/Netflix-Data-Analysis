# 🎬 Netflix Movies Data Analysis

## 📌 About This Project
This project is an Exploratory Data Analysis (EDA) of a Netflix movies dataset. The goal of this project is to analyze the data and find interesting trends, such as the most popular genres, movie popularity over the years, and rating distributions.

## 🛠️ Tools & Libraries Used
* **Python**
* **Pandas** (Data Cleaning & Manipulation)
* **Seaborn & Matplotlib** (Data Visualization)
* **Jupyter Notebook**
